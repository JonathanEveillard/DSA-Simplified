# Data Structures and Algorithms Made Easy

## Overview

This repository is a structured collection of exercises and implementations designed to build a deep understanding of Data Structures and Algorithms (DSA).

The primary source for this project is **Open Data Structures (ODS) – Java Edition**, supplemented with secondary resources to reinforce concepts and provide alternative explanations.

The focus is on **learning by doing**—not memorizing, not copying, but building and understanding from the ground up.

## Learning Approach

Each topic follows a consistent process:

1. Study the concept from the primary source (**ODS**)
2. Reinforce understanding using secondary resources
3. Implement the data structure independently
4. Solve exercises related to the topic
5. Review mistakes and refine understanding

---

## Exercises Organization

Each folder inside `/exercises/` represents a specific topic.

Example:

exercises/BinaryTree/


Inside each topic:
- `java/` → Java implementations and exercises  
- `c/` → C implementations and exercises  

This dual-language approach helps strengthen understanding of both high-level structure and low-level memory behavior.

## AI Usage Policy

AI is used strictly as a **learning support tool**.

### Allowed Uses
- Explaining concepts  
- Providing hints and guiding questions  
- Assisting with debugging  
- Reviewing code and suggesting improvements  

### Not Allowed
- Copying full solutions without understanding  
- Using AI to bypass problem-solving  
- Submitting AI-generated work as original  

All work in this repository reflects personal understanding and effort.
